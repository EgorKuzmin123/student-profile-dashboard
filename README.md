## 📑 О проекте
**Бизнес-цель MVP:** Переход от реактивной работы деканатов (отчисление по факту заваленной сессии) к проактивной — раннему выявлению проблемных студентов за 4–8 недель до критической ситуации. 

Дашборд агрегирует синтетические данные об успеваемости, активности в LMS (система дистанционного обучения) и реакции на карьерные стажировки. 

## 🧠 Проверяемые гипотезы
1. Снижение активности в LMS является самым ранним маркером риска, проявляющимся до появления первых долгов.
2. Студенты в зоне высокого риска отчисления демонстрируют нулевую конверсию в открытие офферов стажировок (маркер потери мотивации).

## 📊 Композитный Индекс Риска (Goodhart's Law)
Чтобы студенты не могли «взломать» метрику (Закон Гудхарта) путем банального накликивания страниц в LMS, спроектирован взвешенный индекс риска $R$:

`R = (0.4 * D) + (0.3 * A) + (0.3 * L)`
Где:
* `D` — Доля задолженностей (Твердая академическая метрика).
* `A` — Падение посещаемости (Относительно средней по группе).
* `L` — Падение активности в LMS (Относительно собственной исторической нормы).

## 🛠 Техническая реализация и архитектура
BI-система работает на гибридной модели данных («Снежинка» с использованием `LEFT JOIN`), чтобы избежать проблемы декартова произведения при пересечении фактовых таблиц логов. 

### Генерация синтетических данных
Скрипт `data_generator.py` генерирует выборку на 500 студентов за 12 недель. В алгоритм математически заложены 3 поведенческих паттерна:
- `80%` — Нормальное обучение.
- `15%` — Плавное увядание (рост долгов).
- `5%` — Резкий обвал (остановка активности на 8-й неделе).


### Инструкция по локальному развертыванию
Если вы хотите сгенерировать данные самостоятельно:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_АККАУНТ/ВАШ_РЕПОЗИТОРИЙ.git

```
2. Установите необходимые библиотеки Python:
   ```bash
   pip install pandas numpy
   ```

3. Запустите скрипт генерации:
   ```bash
   python data_generator.py
   ```
   *На выходе появятся 4 файла: `students.csv`, `academic.csv`, `lms.csv`, `offers.csv`. Данные полностью готовы для загрузки в любую BI-систему.*
